# INSTALL & PLAY

$ git clone --recursive https://github.com/hugeping/Brus-16-games/
$ cd Brus-16-games/Brus-16
$ make
$ cd ..
$ ./python3 ./gerion_make.py && ./brus16 ./gerion.bin

# STORY

2126. An interstellar object—the asteroid Geryon—has been detected
in the solar system. Its trajectory intersects Earth's orbit.
A spacecraft is dispatched to the object.

# CONTROLS

- A - laser
- B - radar
- ARROWS - move
